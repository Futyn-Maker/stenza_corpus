На данный момент поддерживается четыре формата запросов и, соответственно, четыре режима поиска:

  1. Введите любое слово (без кавычек), чтобы найти употребления всех словоформ, которые соответствуют данной лексеме.
    • дом — найдёт предложения, в которых есть словоформы "дом", "дома", "домами" и др.
    • дому — найдёт те же самые предложения (то есть поиск осуществляется по лемме).
  2. Введите слово в "двойных" кавычках, чтобы найти предложения с употреблением только заданной словоформы.
    • "дому" — найдёт все предложения, в которых употреблена словоформа "дому".
    • "пошёл" — найдёт предложения с употреблением словоформы "пошёл".
  3. Введите частиречный тег (в формате Universal Dependencies), чтобы найти все предложения с употреблением слов заданной части речи. Доступны следующие теги:
    • NOUN — существительное;
    • ADJ — прилагательное;
    • VERB — глагол;
    • ADV — наречие;
    • PROPN — имя собственное;
    • PRON — местоимение (любой разряд, кроме детерминативов);
    • DET — определительное местоимение ("этот", "тот" и другие);
    • NUM — числительное (в т.ч. и цифровая запись);
    • CCONJ — сочинительный союз;
    • SCONJ — подчинительный союз;
    • ADP — предлог;
    • PART — частица;
    • AUX — вспомогательный глагол (например, формы глагола "быть").
  4. Введите лемма+POS-тег, чтобы найти предложения с употреблениями всех словоформ, которые соответствуют заданной лексеме с заданной частью речи.
    • и+CCONJ — найдёт все предложения, в котором употреблен сочинительный союз "и".
    • и+PART — найдёт все предложения, в которых употреблена аддитивная частица "и".

Корпус также поддерживает поиск N-грамм любой длины — для этого в поисковом запросе нужно написать несколько токенов через пробел. Каждый токен может задаваться любым из четырёх способов, описанных выше, их можно комбинировать. Например, чтобы найти все сочетания "прилагательные + существительное дом в любой форме",, можно написать: ADJ дом — найдёт "большой дом", "маленькому дому" и другие.
